# Tuanzi Codex Pet

Tuanzi is a hand-drawn custom pet for Codex.

This pet was drawn frame by frame by the author. It is not an AI-generated pet.

## Installation on Windows

1. Copy the `tuanzi` folder to:

   ```text
   C:\Users\<your username>\.codex\pets\
   ```

2. Open Codex.
3. Go to Settings -> Appearance -> Pets.
4. Refresh custom pets.
5. Select Tuanzi.
